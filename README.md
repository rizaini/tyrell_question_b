# SQL Query Optimization

This repository contains the optimization of a SQL query to improve its performance. The original query and the optimized query are provided, along with an explanation of the changes made.

## Files
- `original_query.sql`: The original SQL query.
- `optimized_query.sql`: The optimized SQL query.
- `suggestion.md`: Detailed explanation of the optimizations.

## Original Query
The original query can be found in [`original_query.sql`](original_query.sql).

## Optimized Query
The optimized query can be found in [`optimized_query.sql`](optimized_query.sql).

## Suggestion
